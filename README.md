## 避坑指南：这些细节注册前先知道

**坑一：单笔低于 10 美金的小额消费**

有用户反馈，单笔低于 10 美金的交易，会被额外扣 0.1 美金左右的小额处理费。如果你打算用它绑一堆月费 5 美金以内的便宜订阅，累计下来会有点肉疼。大额消费没这个问题。

**坑二：充值网络选错**

PokePay 支持 TRC20、ERC20、Polygon、AVAX C-Chain 等多个网络，但官方文档对每个币种具体支持哪些网络的说明不够透明。**提币前一定先在 App 内确认对应币种的充值地址和适用网络**，选错网络钱可能找不回来。TRC20 一般是最稳的，手续费也最低。

**坑三：Steam 等部分平台不支持**

PokePay 对 Steam、Blizzard、EA 等游戏平台的支付支持不太稳定，有用户反馈扣款失败或者被拒。如果你的主要用途是买游戏，这块要提前心里有数。Netflix、Spotify、ChatGPT Plus、Amazon、Google Ads 这些主流场景是稳的。

**坑四：IP 与卡片地区不一致**

用 PokePay 订 ChatGPT Plus 这类对地区敏感的服务时，如果你的 IP 和卡片签发地（香港）不一致，可能会触发风控。建议访问目标服务时保持 IP 稳定，不要频繁切换地区。

**坑五：实体卡跨境费**

虚拟卡通过专属邀请渠道注册可以享受 0% 跨境费，但实体卡的非香港消费会有 1%-2% 的跨境手续费。如果你主要在国外刷卡，这笔费用要算进成本里。

**Q：邀请码必须填吗？不填会怎样？**

**Q：邀请码填错了能改吗？**

注册时绑定的邀请关系一般不能改。所以建议直接通过 👉 [带邀请码的注册链接](https://bit.ly/PoKePay) 进入，让系统自动识别，不用手动输，避免填错。

可以。PokePay 支持大陆身份证 KYC，也支持护照（护照实名姓名填英文）。

**Q：优惠券什么时候到账？**

注册成功后优惠券会自动发到账户里，一般在"申请卡片"页面能看到并选择使用，不需要手动兑换。

**Q：虚拟卡和实体卡能同时开吗？**

可以。两张卡是独立的，各自有开卡费，但可以共用一个账户余额。实体卡需要先开通虚拟卡之后才能申请。

**Q：充值用什么网络最稳？**

USDT 用 TRC20 最稳，手续费最低，到账也快。ERC20 也行但贵。其他网络提币前一定先在 App 内确认。

**Q：被封号或限制地区有哪些？**

PokePay 不支持美国、俄罗斯、伊拉克、伊朗、朝鲜等地区（完整名单官网有列）。中国大陆不在限制名单内，可以正常使用。
